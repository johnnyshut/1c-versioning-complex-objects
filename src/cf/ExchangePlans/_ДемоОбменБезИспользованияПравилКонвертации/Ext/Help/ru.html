Этот план обмена предназначен для демонстрации обмена данными между однотипными программами Библиотеки стандартных подсистем без использования правил конвертации.

План обмена определяет состав объектов метаданных, которые участвуют в обмене данными.
Использование правил регистрации позволяет настраивать фильтрацию данных при обмене. Настройка ограничения миграции данных выполняется на закладке "Настройка ограничения миграции данных":

  • параметр "Дата начала выгрузки документов" используется для ограничения выборки документов заданной датой; будут выгружены только документы, дата которых больше или равна установленной даты, например, с начала года;
  • признак "Отбор по организациям" используется для включения ограничения миграции данных по заданному списку организаций.


Для элемента справочника Организации регистрация изменений выполняется, если элемент принадлежит множеству элементов из соответствующей таблицы в узле.

Для документа регистрация изменений выполняется, если:

  • дата документа больше или равна параметру "Дата начала выгрузки документов" в узле;
  • и организация входит в число указанных организаций при узле плана обмена (при условии установки признака "Отбор по организациям").

Также план обмена демонстрирует работу дат запрета загрузки (подсистема Даты запрета изменения) для следующих типов данных:
Документ "Демо: Поступление товаров"
Документ "Демо: Реализация товаров"
Регистр сведений "Курсы валют"
